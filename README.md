# GDG_2K26

A React + Vite based web application built for GDG 2K26.

## 🚀 Tech Stack
- React
- Vite
- JavaScript
- CSS

## 📂 Project Structure
src/
components/
pages/
assets/

## ⚙️ Installation

Clone the repository

git clone https://github.com/PriyanshusinghPanda/GDG_2K26.git

Install dependencies

npm install

Run the development server

npm run dev

## 📌 Features
- Interactive UI
- Quiz Portal
- Concept learning pages
- Responsive design
